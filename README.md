# Pong Game

A classic Pong game built with HTML5 Canvas and JavaScript.

## Features

- Play against AI opponent
- Sound effects for ball bounces, scoring, and game over
- Arcade-style graphics
- First to 10 points wins

## How to Play

1. Open `index.html` in your web browser
2. Click on the canvas to start the game
3. Use **Arrow Up** and **Arrow Down** keys to move your paddle
4. Score points by getting the ball past your opponent's paddle
5. First player to reach 10 points wins!

## Controls

- **Arrow Up**: Move paddle up
- **Arrow Down**: Move paddle down
- **Mouse Click**: Start game / Play again

## Files

- `index.html` - Main HTML file
- `game.js` - Game logic and rendering
- `style.css` - Styling
- `sounds/` - Sound effects directory
  - `bounce-sound.wav` - Ball bounce sound
  - `score_sound.wav` - Scoring sound
  - `game_over.wav` - Game over sound

## Technologies Used

- HTML5 Canvas
- JavaScript (ES6)
- Web Audio API

## Installation

No installation required! Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/YOUR_USERNAME/pong.git
cd pong
open index.html
```
